# Flight Price Monitor ✈️

Automação criada com n8n para monitorar preços de passagens aéreas.

## Tecnologias

- n8n
- Docker
- SQLite
- Telegram

## Funcionalidades

✔ consulta preço de voos  
✔ salva histórico  
✔ detecta queda de preço  
✔ envia alerta automático

## Como rodar

docker compose up

Acesse:

http://localhost:5678
